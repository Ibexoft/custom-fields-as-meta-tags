    • Custom Fields as Meta Tags WordPress Plugin

    Add custom fields of any post as meta tag in head section of WordPress post. It is useful for SEO purposes when you want to add any data as meta tags in head section of that post. You don't need any sophisticated plugin or learn coding to add meta tags for your keywords. Just add custom fields in your post and this plugin will automatically add those custom fields as meta tags in your post header.

    Project maintained by mjawaids Hosted on GitHub Pages — Theme by mattgraham

    Installation

    1. Upload custom-fields-as-meta-tags folder to the /wp-content/plugins/ directory
    2. Activate the plugin through the 'Plugins' menu in WordPress
